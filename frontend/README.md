# Research Agent UI

This is a React frontend for the Research Agent multi-agent research assistant.

## Setup

1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev`

## Backend

The frontend expects a backend wrapper on `http://localhost:8000`.
Run the backend from the repository root with:

```bash
pip install -r requirements.txt
python backend.py
```

Then open the React app URL shown by Vite.
